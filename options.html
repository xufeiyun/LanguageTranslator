Basic Operations
  • Translate Text automatically by selecting text
  • Translate Text automatically by double-clicking text
  • Translate Text automatically by typing text
More Operations
  • Open Wiki Page for Source Text
  • Pronunciation of text
  • Copy Selected Text automatically if the [Enable Copy Text] feature enabled
  • Open home page of this extension
Basic Settings Options with BLUE TITLE will be enabled by default.
Other Settings
MOST IMPORTANT FEATURE

Translate text between English and Chinese Simplified by default.

  • ONE STEP ONLY to Translate Text via selecting text on webpage!

Yeah, installing this Google extension, then you get results immediately ONLY by ONE STEP:

1. Select a word, phrase or sentence.

It automatically translates the text by presenting a popup dialog in top-right corner in the current page.

Just it.

More Features
  1. Type texts directly.
  2. Translate texts automatically after your typed it (ONE AWESOME FEATURE).
  3. Translate texts manually by shortcut (Ctrl+Enter).
  4. Translate texts manually by clicking the Translate button.
  5. Translate texts directly after you pasted it (Ctrl + V).
  6. Open a wiki page for the Source Text.
  7. Pronunciation for texts.
  8. Collapse or Expand the Popup Dialog.
  9. Disable or Enable Text Translation for current page.
  10. Close the Popup Dialog for current page.
